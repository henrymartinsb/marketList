# marketList

A simple market list use React and Local Storage

<br>
<br>

## ⏭ Demo

Live Preview: https://homermarketlist.vercel.app

<br>
<br>

## 💻 Run Locally

Clone the project

```bash
  git clone https://github.com/henrymartinsb/marketlist
```

Go to the project directory

```bash
  cd marketlist
```

Install dependencies

```bash
  npm install
  yarn install
```

Start the server

```bash
  npm run dev
  yarn dev
```
